# Bible Message Generator
>
## Description 
> #### The Bible Message Generator Application is an application that randomly displays quotes from the Bible.
> #### The App is interactive enough to take your name and and address the randomly selected Bible quote with your Name.
> #### This Application was built to inspire people and get them familiar with some Quotes from the Bible.
>
## Feature 
> #### The Application is strictly Javascript.
>
## How to Use
> #### Ensure you have the latest Node Version installed on your system.
> #### Copy the folder or clone the repo to your system and run on VSCode.
> #### Run the Application with the Command node main.js and you are good to go.
